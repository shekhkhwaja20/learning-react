## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 📦
- npm or yarn 📦
- Basic JavaScript knowledge 💡

### Installation

```bash
# Clone the repository
git clone https://github.com/shekhkhwaja20/learning-react.git

# Navigate to project
cd learning-react

# Install dependencies for a specific project
cd learning && npm install
```

### Running Projects

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
